## S-01. 🏠 홈 화면

> 앱을 열면 가장 먼저 보이는 화면. 캐릭터 상태, 오늘의 퀘스트, XP 진행 상황, 보유 티켓을 한눈에 확인.
> 

![S-01 홈 화면](https://i.imgur.com/yms6b4s.png)

S-01 홈 화면

**주요 컴포넌트**

- 상단 상태바: 🔥 스트릭 일수 / 히든 프로필 단계 / 🎟 보유 티켓
- 캐릭터 영역: Lottie 애니메이션 + 말풍선 (시간대별 멘트 변화)
- EXP 진행 바: 현재 레벨 경험치 시각화
- 오늘의 퀘스트 카드: 완료 ✅ / 진행 중 [▶ 시작] 상태 구분
- 가챠 배너: 보유 티켓 수 + 현재 확률 상승 상태(srBonus) 표시

---

## S-02. 📚 학습 화면 (AI 퀴즈)

> OX / 객관식 / 빈칸 채우기 / 상황 카드 4가지 유형을 동일한 UI 틀로 제공. 정답/오답 시 Lottie 연출과 해설 패널 슬라이드업.
> 

![S-02 학습 화면](https://i.imgur.com/mhoLwN3.png)

S-02 학습 화면

**주요 컴포넌트**

- 진행 바: **총 10문제** 대비 현재 진행률
- 오답 카운터 UI: 현재 오답 수 표시, **3개 이상 틀리면 미션 실패** 안내
- 캐릭터 말풍선으로 문제 출제 (문제 유형별 다른 캐릭터)
- 선택지 카드: 선택 시 테두리 강조 → 0.5초 후 정답 판정
- 정답 해설 패널: 하단에서 슬라이드업, 쉬운 언어로 개념 설명
- 결과 요약 패널: `정답 수 / 10`, `오답 수`, `통과 여부`, `펫 희귀도 보너스 XP` 표시

---

## S-03. 💬 GPT 프롬프트 실습 화면

> 아이가 실제 AI와 대화하며 프롬프트 작성법을 배우는 화면. ML Kit 실시간 개인정보 감지 + 자동 채점 포함.
> 

![S-03 GPT 실습 화면](https://i.imgur.com/kTPk1dS.png)

S-03 GPT 실습 화면

**주요 컴포넌트**

- 오늘의 미션 배너: 오늘 연습할 주제 제시
- 채팅 UI: AI 말풍선(회색, 좌) / 사용자 말풍선(보라, 우)
- 🛡 안전모드 배지: ML Kit 개인정보 감지 ON 상태 표시
- 프롬프트 채점 결과: 구체성/안전성/완성도 3항목 바차트
- 힌트 버튼: 예시 프롬프트 BottomSheet 표시

---

## S-04. 🎰 가챠 뽑기 화면

> 티켓을 소모해 펫 캐릭터 카드를 랜덤 획득. 등급별 빛 연출 + 확률 상승 시스템 표시.
> 

![S-04 가챠 뽑기 화면](https://i.imgur.com/Wl4R7t9.png)

S-04 가챠 뽑기 화면

**주요 컴포넌트**

- 확률 정보 바: 현재 뽑기 횟수 구간과 티켓 종류 기준 등급별 확률 공개
- 카드 뽑기 영역: 별빛 배경 + 글로우 링 + 3D 카드 뒤집기 애니메이션
- 등급별 빛 연출: 커먼(흰) / 레어(파랑) / SR(보라) / 레전드(금빛 폭발)
- 확률 상승 정보: 누적 뽑기 횟수 / `srMissCount` / `srBonus` / 중복 조각 수
- 교환소 버튼: 조각 4개 이상 시 활성화

---

## S-05. 🏆 수집 갤러리 & 랭킹

> 획득한 AI 몬스터 도감 + 주간 EXP 기준 친구 랭킹. 미획득 캐릭터는 실루엣으로 표시.
> 

![S-05 수집 갤러리](https://i.imgur.com/YwGciU1.png)

S-05 수집 갤러리

**주요 컴포넌트**

- 탭: [📖 도감] / [🏅 랭킹] 전환
- 수집 현황 바: 전체 22종 대비 획득 수 / 완성률 %
- 등급 필터 칩: 전체 / 커먼 / 레어 / SR / 레전드
- 4열 그리드: 획득 캐릭터 컬러 / 미획득 흑백 실루엣 + 🔒
- 캐릭터 상세 카드: 이름 / 등급 / 획득일 / 특기 / 중복 조각 수

---

## S-06. 👨‍👩‍👧 학부모 대시보드

> 부모 Google 로그인으로 진입하는 전용 화면. 자녀의 오늘 학습 요약 + 주간 EXP 차트 + 개인정보 안전 리포트.
> 

![S-06 학부모 대시보드](https://i.imgur.com/TuMrBNN.png)

S-06 학부모 대시보드

**주요 컴포넌트**

- 환영 카드: 자녀 이름 / 레벨 / 스트릭 / 오늘 퀘스트 달성 수
- 오늘 요약 4칸: 퀘스트 완료 / 획득 XP / 뽑기 사용 / 스트릭 상태
- 주간 XP 바차트: 요일별 학습량 막대 그래프
- 🛡 개인정보 보호 리포트: 개인정보 입력 시도 횟수, 필터 작동 횟수
- 알림 설정: 미학습 알림 / 개인정보 경고 / 주간 리포트 / 레벨업 축하 ON/OFF 토글
- 자주 틀리는 미션 요약 카드

## 🔗 화면 전환 흐름

```jsx
[스플래시] → [온보딩(최초)] → [홈 S-01]
   홈 S-01
    ├── 퀘스트 [시작] → [학습 S-02]
    │                      ├── 오답 0~2개 → 미션 통과 → 결과 화면(Perfect/펫 희귀도 보너스 표시) → 홈 복귀
    │                      └── 오답 3개 이상 → 미션 실패 → 재도전 유도
    ├── 하단탭 [AI실습] → [GPT 실습 S-03] → 완료 후 홈 복귀
    ├── 하단탭 [수집] → [갤러리&랭킹 S-05]
    ├── 가챠 배너 탭 → [가챠 S-04] → 완료 후 갤러리 이동
    └── 부모 계정 메뉴 → Google 로그인 상태 확인 → [학부모 대시보드 S-06]
```
